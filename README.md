# 📐 Math PDF Assistant

A modern, minimalist AI assistant designed specifically for analyzing mathematical documents and solving quiz problems. Built with a focus on ephemeral data security and high-accuracy LaTeX rendering.

![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=for-the-badge&logo=Streamlit&logoColor=white)
![HuggingFace](https://img.shields.io/badge/%F0%9F%A4%97%20Hugging%20Face-Spaces-yellow?style=for-the-badge)

## ✨ Features
- **Ephemeral Context:** For maximum security and privacy, document data is purged from memory after every single prompt.
- **LaTeX Support:** Specialized in rendering complex calculus, integrals, and algebraic equations beautifully.
- **Strict Guardrails:** Optimized to refuse off-topic queries, focusing entirely on document analysis.
- **Modern UI:** Gemini-inspired floating dock interface with smooth CSS transitions.
